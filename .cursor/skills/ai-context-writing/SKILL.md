# AI Context Writing Skill

# Rules
- Paragraph structured content must be used very sparingly and **ONLY** to describe the context, **NOT** the specific rules.
- Rules must **ALWAYS** be in list format and as concise as possible while still cleartly describing the rule.
- Example references must be generic placeholder that convey the intent of the rules and not specific to any current implementation.

# **REQUIRED** End Review

The following must be reviewed after finishing the task before the task can be considered complete:

- Review for any glaring issues.
