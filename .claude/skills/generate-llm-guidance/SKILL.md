# Generate LLM Guidance

You are tasked to review the code in the referenced directory and generate an `llm.md` file that is geared **SPECIFICALLY** for llm consumption on how to use the code in the directory. If an existing `llm.md` file existing, **IGNORE** that file and just write over it.
